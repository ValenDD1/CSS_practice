Funciones

en css existen la propiedad @function pero por el momento no hay compatibilidad con todos los navegadores ,pero se veria de esta forma

Ejemplos

syntaxis de la funcion

@function --color(--value){ result: var(--value); }

syntaxis de la funcion con tipo de dato

@function --color( --value type( <color>)){ result: "es color " + var(--value); }

syntaxis de la funcion con tipo de dato

@function --fuente-responsiva(){ result: 1rem ; @media (width >600px) { result:1.5rem; } }
