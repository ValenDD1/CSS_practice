Selectores CSS

aprenderemos a usar los selectores en css ,la estructura que tiene css mediante los selectores clases ids pseudo clases y elementos ademas de atributos

selector id class [atributo] pseudoclase pseudoelemento { propiedad: valor; }
nos damos cuenta que en css hay varias categorias de selectores
  • Selector: seleccionamos las etiquetas globales
  • id: seleccionamos todos los elementos con esa id (ojo la id debe ser unica)
  • class: seleccionamos todos los elementos con esa clase
